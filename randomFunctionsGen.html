Definitions

Gaussian process is a stochastic process where any finite number of random variables have a joint Gaussian distribution.

A Gaussian process is completely specified by its mean function \(m(x)\) and covariance function \(k(x,x')\). They are defined by

\[m(x) = \mathcal{E}[f(x)],\]

\[ k(x,x') = \mathcal{E}[(f(x) − m(x))(f(x') − m(x'))],\]

and will write the Gaussian process as \( f(x) \sim GP(m(x), k(x, x'))\).

Squared exponential

\[ cov(f(x_p), f (x_q)) = k(x_p , x_q) = \exp\Big(−\frac{1}{2} |x_p − x_q |\Big)^2.\]

squared.exp <- function(x,y,l=1) {
  exp(-0.5* (norm(x-y, type="2")/l)^2)
}

Sigma <- function(x1, x2, l=1) {
  n1 <- length(x1)
  n2 <- length(x2)
  S <- matrix(rep(0, n1*n2), nrow=n1)
  for (i in 1:n1) {
    for (j in 1:n2) {
      S[i,j] <- squared.exp(x1[i],x2[j],l)
    }
  }
  S
}
